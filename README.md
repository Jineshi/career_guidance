Overview

The Career Guidance Web Application is a web-based platform designed to help students and early professionals explore suitable career paths based on their interests, skills, and goals.
The application connects users with professionals, provides personalized recommendations, and helps users make informed career decisions.

This project focuses on user-centric design, secure authentication, and scalable architecture.
